# ユーザーフィードバック（2026-03-08）

## 復習・履歴

- 間違えた問題の履歴で、正解だけでなく **選択肢の全文（例: A.拒否権）** も表示してほしい。どう間違えたかを確認したい
- 正解「C」だけでなく **問題文も一緒に載せてほしい**（勉強になる）
- 選択肢が「A, B, C, D」だけでなく **「A 拒否権」のように内容付き** だとより覚えやすい

## ゲームプレイ中のUX

- 答えを選んだ後、**即座に○×と正解・解説を表示** してほしい（解きながら学べる）
- ゲーム開始前に **遊び方の短い説明**（チュートリアル）がほしい

## モード・設定

- **1人プレイモード** があるといい（対戦相手なしで練習）
- **難易度選択**（レベル）ができるといい
